# Karmatical's TextBlocks

A text-based project editor for Scratch projects.

This project is inspired by (but is **not** a fork of) the original **Tosh Editor** ([GitHub](https://github.com/tjvr/tosh2/), [Website](https://tosh.blob.codes/app/)). Tosh currently only supports the older SB2 format and has not been updated in several years, which prompted me to develop a modern alternative.

This project is **open-source** and supports custom **Javascript-based Addons** (equivalent to Extensions).

* Management interfaces for **Sprites, Costumes, and Sounds**.

## üöß Known Limitations / Future Roadmap

The following features are currently *not* supported, but may be added in future updates:

* **Legacy Scratch 2.0 (SB2) Format** support.
* Support for **Most Official Scratch Extensions** (Only the Pen extension works currently).
* Integration with **Turbowarp** (and Turbowarp Extensions).
* Dedicated **Desktop Application** (Currently runs only on GitPages).
* **Cloud Variables** (Variables can be manipulated offline, but will not connect to a server).
